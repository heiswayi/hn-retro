## Getting Started

To start using this theme, simply clone the repository and install the dependencies:

```bash
git clone https://github.com/yourusername/eleventy-theme.git
cd eleventy-theme
npm install
npm start
```

This will start a local development server at `http://localhost:8080`.

## Features

*   **Fast**: Built on Eleventy, a static site generator known for its speed.
*   **Simple**: No complex build pipelines or heavy frameworks.
*   **Customizable**: Easy to tweak styles using SASS.

Enjoy building your site!
